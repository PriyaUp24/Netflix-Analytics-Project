# Netflix Content Analysis

## Project Overview
This project analyzes Netflix content data to understand content distribution,
growth trends, genre popularity, and regional contributions using Python.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook

## Key Analysis
- Data cleaning and preprocessing
- Movies vs TV Shows distribution
- Content growth over time
- Country-wise content contribution
- Genre and rating analysis

## Key Insights
- Movies dominate Netflix’s content library
- Significant increase in content after 2015
- USA and India are top content-producing countries
- Drama and International genres are most common
- Majority of content targets mature and teen audiences

## Dataset
Source: Kaggle – Netflix Titles Dataset
